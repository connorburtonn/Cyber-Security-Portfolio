# Network Design - Northbridge Ltd
## Overview:
This project outlines the proposed network architecture for the organisations IT environment. The design aims to provide a secure, scalable and reliable environment that supports business operations while following cybersecurity best practices.

This network will be developed throughout the portfolio as additional technologies and security controls are implemented.



## Objectives:

The network has been designed to:

- Support approximately 50 employees
- Provide secure access to company resources
- Separate critical systems from user devices
- Support future expansion
- Enable centralised management and monitoring
- Protect sensitive business data



## Proposed Infrastructure:

The planned environment will include:

- Internet connection
- Router
- Firewall
- Network switch
- Windows Server (Active Directory Domain Controller)
- File Server
- Employee Windows 11 workstations
- Ubuntu workstation for Linux administration
- Wireless access for approved devices



## Planned Network Layout:

The environment will eventually consist of:

- Internal corporate network
- Server segment
- User workstation segment
- Internet connection protected by a firewall

A detailed network diagram will be added as the lab environment is developed.



## Security Considerations:

The network will incorporate the following security principles:

- Principle of Least Privilege
- Network segmentation
- Centralised authentication
- Strong password policies
- Multi-factor authentication where appropriate
- Regular backups
- System logging and monitoring



## Future Improvements:

The following technologies will be added during later stages of the project:

- Active Directory
- Group Policy
- DNS
- DHCP
- Vulnerability scanning
- Security Information and Event Management (SIEM)
- Incident response procedures
- Threat monitoring
- Security hardening



## Status:

Current status: Planning phase

The network will be implemented practically during the Network+ and Security+ learning journey.
