# Asset Inventory – Northbridge Ltd

## Overview:
This document lists key IT and business assets within the organisation. It is used to support risk assessment, security controls, and incident response planning.



## End User Devices:
- Windows 11 laptops (employee workstations)
- Administrative workstations (IT staff)
- Remote employee laptops (VPN-enabled access)



## Servers & Core Systems:
- Windows Server (Active Directory Domain Controller)
- File Server (centralised document storage)
- Backup Server (planned)
- Authentication services (Active Directory / Entra ID simulation)



## Network Infrastructure:
- Corporate LAN (internal network)
- Firewall (virtualised)
- Network switches (logical in lab environment)
- Router (internet gateway simulation)
- DNS and DHCP services (hosted on Windows Server)



## Cloud Services (Simulated / Planned):
- Microsoft 365 (email and collaboration)
- OneDrive / SharePoint (file sharing simulation)
- SIEM platform (Microsoft Sentinel)



## Business-Critical Data:

### High Sensitivity:
- Payroll information
- Employee personal data
- Financial records
- Client contracts

### Medium Sensitivity:
- Internal emails
- Operational documents
- Project files

### Low Sensitivity:
- Public marketing materials
- General company information



## Security-Critical Assets:
These are the most important systems in the organisation:

- Active Directory Domain Controller
- File server containing HR and Finance data
- Email system (phishing entry point)
- Privileged admin accounts
- Backup systems



## Objective:
The purpose of this asset inventory is to:
- Identify critical systems and data
- Support risk assessment and vulnerability management
- Prioritise security controls based on business impact
