# Company overview - Northbridge Ltd
This document describes the fictional organisation.

## Purpose:
Northbridge Services Ltd is a simulated UK-based organisation created to replicate a real-world enterprise environment for cybersecurity portfolio development.

This environment is used to demonstrate practical skills in security operations, network administration, risk management, and incident response.

## Business Type:
Mid-sized professional services company (approximately 50–100 employees).

## Departments:
- Executive Management
- Finance
- Human Resources
- Operations
- Sales
- IT Support

## Environment:
- Windows-based corporate network
- Active Directory domain (planned implementation)
- Centralised file storage system
- Security monitoring and logging (SIEM simulation)
- Endpoint devices (Windows 11 workstations)

## Assumptions:
- All data and users are fictional
- No production systems are used
- The environment is isolated and used for educational purposes only
- Security controls are implemented in a lab setting

## Scope:
This environment simulates enterprise IT infrastructure for the purpose of:
- Security monitoring
- Threat detection and response
- Vulnerability management
- Identity and access control

## Objective:
To design, build, secure, and monitor a realistic enterprise environment as part of a structured cybersecurity learning and portfolio project.

## Evolution:
This environment will be progressively developed to include advanced security monitoring, incident response simulations, and infrastructure hardening over time.
